## What is this?

A semester-wise CGPA calculator built specifically for **Lok Jagruti Kendra University (LJKU)** B.E. students. No spreadsheets. No mental math. No asking your topper friend to calculate it for you at 11 PM before results.

Just enter your SPI and credits for each semester, hit **Calculate**, and receive your CGPA with a satisfying animated ring — because if the number is bad, it should at least *look* good.

---

## Features

- 📅 **1–8 semester support** — from the blissfully unaware Sem 1 you to the existentially exhausted Sem 8 you
- 🔢 **Pre-filled default credits** — because who remembers how many credits Sem 3 had
- 🏆 **Auto-classification** — finds out if you're "Pass Class" or "First Class with Distinction" (and everything painful in between)

---

## Grading Scale

| CGPA | Classification |
|------|---------------|
| < 5.00 | Pass Class *(you passed, that's the win)* |
| 5.00 – 5.49 | Second Class |
| 5.50 – 5.99 | Higher Second Class |
| 6.00 – 6.99 | First Class |
| ≥ 7.00 | First Class with Distinction 🎉 |
